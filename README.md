# Netflix-Project
## Overview

### The project answers six questions on the Netflix Movies and TV shows dataset. The dataset was first assessed, cleaned and then analused and visulaised to get insights.

## Prerequisites

- Python 3.7
- Matplotlib
- Seaborn
- Pandas
- Numpy
- Plotly
